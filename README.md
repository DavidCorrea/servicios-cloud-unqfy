# UNQfy

## Requerimientos
- Node (14.4.0)

## Instalación
- Clonar el repositorio
- `npm install`

## Comandos (Proyecto)
- Para correr los tests: `npm run test`
- Para correr el linter: `npm run lint`

## Comandos (UNQfy)
- `addArtist name "NAME" country "COUNTRY"`
- `getArtist id "ID"`
